# LinkedInNetworkingHelper

I made an app to help me with finding people to reach out to about referrals or company information for specific jobs. Make sure to update the simplify page or just code up a scraper to keep pulling it from github. The app scores jobs by how important it is and how likely I am to find success with reaching out, then there is another file that uses google x ray searches with the relevant linkedin information to try and find people that match - it is very bad at this however and linkedin is annoyingly unbotifiable!

Make sure to create a .env file with your groq/anthropic api key
